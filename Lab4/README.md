##Описание проекта

Проект реализует:
1) Оператор печати std::tuple при помощи рекурсивных шаблонов
2) Класс, делающий возможным потокувую работу c CSV файлами. 

## Инструкция сборки
1) Открыть командную строку в данной папке
2) Прописать

```
cmake -S . -B build
```
Затем
```
cmake --build build --config Debug
